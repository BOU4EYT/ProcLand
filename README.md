# ProcLand

ProcLand is a **single-file procedural idle strategy game** built in plain HTML/CSS/JavaScript.
Open `main.html` in any modern browser to play.

## What's in the game

- **Seeded procedural worlds** with biomes, zone traits, risk/reward tuning, and escalating frontier tiers.
- **Idle expedition gameplay** that continuously gathers resources from a selected zone.
- **Resource economy** with gathering, crafting, production buildings, and market trading.
- **Automation controls** for targeting, crafting, building, and research workflows.
- **Contracts + random encounters** for dynamic goals and run variance.
- **Meta progression** through Echo Shards and prestige-like World Jump resets.
- **Autosave + offline progress** to support true idle play.

## How to run

1. Clone this repo.
2. Open `main.html` directly in your browser.

No build step, no dependencies, no framework required.

## Gameplay loop

1. Explore procedural zones and gather starter resources.
2. Craft tools/alloys/drones and build your production base.
3. Complete contracts and invest in research.
4. Enable automation policies to scale efficiently.
5. World Jump to gain shards, then buy permanent meta upgrades.
6. Repeat with new seeds for different strategies and outcomes.

## Notes

- Save data is stored in browser `localStorage` under `procland-infinite-save`.
- The game is intentionally designed for replayability and long idle sessions.
